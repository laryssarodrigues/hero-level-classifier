# Hero Level Classifier

A simple hero level classifier based on experience points (XP).  
This project was developed as part of a coding challenge to practice basic Java concepts.

## Features

- Asks for the hero's name;
- Asks for the amount of experience (XP) with input validation;
- Classifies the hero into different levels based on XP;
- Displays the formatted result.

## Classification Levels

| XP Range | Level |
|-------------|-------|
| < 1000 | Iron |
| 1001 - 2000 | Bronze |
| 2001 - 5000 | Silver |
| 5001 - 7000 | Gold |
| 7001 - 8000 | Platinum |
| 8001 - 9000 | Ascending |
| 9001 - 10000 | Immortal |
| > 10000 | Radiant |

## Usage Example

>>>|HERO LEVEL CLASSIFICATION SYSTEM|<<<
Hero's name: He-Man
Amount of experience (XP): 8500
The hero named He-Man is at the Platinum level.
